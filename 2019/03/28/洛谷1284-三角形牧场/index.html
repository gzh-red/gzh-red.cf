洛谷P1284 三角形牧场

发表于 2019-03-28 | 分类于 洛谷

题目链接

题目描述

和所有人一样，奶牛喜欢变化。它们正在设想新造型的牧场。奶牛建筑师Hei想建造围有漂亮白色栅栏的三角形牧场。她拥有$N$($3≤N≤40$)块木板，每块的长度$L_i$($1≤L_i≤40$)都是整数，她想用所有的木板围成一个三角形使得牧场面积最大。

请帮助Hei小姐构造这样的牧场，并计算出这个最大牧场的面积。

输入输出格式
输入格式：

第$1$行：一个整数N

第$2..N+1$行：每行包含一个整数，即是木板长度。

输出格式：

仅一个整数：最大牧场面积乘以100然后舍尾的结果。如果无法构建，输出-1。

输入输出样例

输出样例#1：

说明
样例解释：692=舍尾后的（100×三角形面积），此三角形为等边三角形，边长为4。

解题步骤

题意理解

就是把N块木板,分成三组,然后让最后组合出来的三角形面积最大.

思路构建

首先看到这道题目,转入脑袋中的思路,应该是DP算法中的最基础的算法背包问题.

然后接着我们分析,每一个木板只能使用一次,所以再次详细地确定这道题目是01背包问题.

确定了是01背包问题后,对于三角形面积,我们可以使用海伦公式进行求值.

关于海伦公式
$$S= \sqrt{p \times (p-a) \times (p-b) \times (p-c)} $$
$$P=\frac{a+b+c}{2}$$

思路探索

然后这道题目还有一个隐含条件就是,一个木板只能放入一个边长之中,所以我们不能单纯地开三个01背包,必须开两个背包同时进行(第三个值,可以通过总重量-其他两个背包求出)

最重要的是,这道题目因为我们只需要判断,这条边的边长,能否被组合出,并不关心时候要用,所以我们可以使用bool数组和位运算,提高效率,节省时间.
