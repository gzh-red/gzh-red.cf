洛谷P1162 填涂颜色

发表于 2019-03-10 | 分类于 洛谷

原题连接
题目描述
由数字0组成的方阵中，有一任意形状闭合圈，闭合圈由数字1构成，围圈时只走上下左右4个方向。现要求把闭合圈内的所有空间都填写成2.例如：$6 \times 6$的方阵$（n=6）$，涂色前和涂色后的方阵如下：

输入输出格式
输入格式：
每组测试数据第一行一个整数n$(1 \le n \le 30)$
接下来$n$行，由$0$和$1$组成的$n \times$的方阵。

方阵内只有一个闭合圈，圈内至少有一个0。

//感谢黄小U饮品指出本题数据和数据格式不一样. 已修改(输入格式)

输出格式：
已经填好数字2的完整方阵。

输入输出样例
输入样例#1：

输出样例#1：

说明
$1 \le n \le 30$

搜索

这道题目其实就是一道普通的搜索题目,大致思路如下所示,首先我们可以发现一个性质,那么就是对于一个在圈内的0而言,那么肯定一个性质满足,那就是它到它最左边,它到它最右边,它到它最上边,它到它最下边,在这四条如今上面,肯定都会有1,不然的话它肯定不在圈内.


话说洛谷的图床还是挺好用的.

然后我们通过这个上面所说的性质,我们就可以进行搜索,将所有和我们找到这个圈内点联通的点,统统都染色,染成2即可.
